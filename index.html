Machine learning

House prediction

This machine learning project focuses on predicting house prices using a dataset containing various features such as the number of bedrooms, square footage, and location. We began by exploring the dataset through visualizations such as histograms, scatter plots, and heatmaps to gain insights into the relationships between variables. After preprocessing the data and performing feature engineering, we trained two models: linear regression and random forest regression. These models were evaluated using the R-squared score, which quantifies the proportion of variance in the target variable that is predictable from the features.

R-squared, or coefficient of determination, measures how well our machine learning models fit the data. A higher R-squared score indicates better prediction accuracy.
